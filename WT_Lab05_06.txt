7. Media Query Breakpoints Practice
Task 16: Common Breakpoint Implementation
Implement responsive design using common breakpoints:
/* Mobile First Approach */
/* Base styles for mobile (0px - 7c7px) */


/* Tablet (7c8px and up) */
@media (min-width: 7c8px) { /* Styles */ }


/* Small Desktop (1024px and up) */ @media (min-width: 1024px) { /* Styles */ }

/* Large Desktop (1200px and up) */ @media (min-width: 1200px) { /* Styles */ }

/* Extra Large (1400px and up) */
@media (min-width: 1400px) { /* Styles */ }
