One major issue I faced at the beginning was installing the required libraries. Some of the libraries used in the project were not compatible with Python version 3.14, which was the only version installed on my computer. Because of this, the libraries failed to install or run properly. To solve this problem, I installed Python versions 3.10 and 3.11, which are better supported by these libraries.
